Cara Hosting gratis menggunakan GitHub Pages

Asalammualaikum wr wb. Sebelumnya saya ucapkan terima kasih kepada anda kaerna sudah menggunjugi website saya, untuk halaman website saya ini saya ingin menjelaskan bagaimana cara hosting hasil codingan website kalian menggunakan GitHub Pages agar bisa diakses oleh orang lain.

Masuk ke website GitHub atau bisa klik link disini https://github.com, setelah itu anda buat akun GitHub jika belum.

Setelah membuat akun anda klik foto profile anda dan buka repository.

Setelah anda klik repository anda bisa klik new atau baru.

setelah itu anda buat nama repository, bebas namanya ingin apa.

Setelah itu anda klik buat repository.

Anda bisa klik menggungah berkas yang sudah ada jika anda sudah membuat file websitenya.

Selanjutnya anda bisa klik Pilih file anda.

Jika sudah anda bisa klik setting atau pengaturan

Jika sudah maka selanjutnya anda bisa klik pagges atau halaman.

Setelah itu anda ubah foldernya ke folder root atau akar.

Jika sudah maka anda bisa klik menyimpan, lalu jika sudah maka kalian bisa ttunngu beberapa saat sambil di refres websitenya.

Terakhir, jika url nya sudah muncul maka hostingnya sudah berakhir dan website anda sudah bisa dilihat oleh audiens.

Mungkin itu yang bisa saya sampaikan, semoga bermanfaat mohon maaf bila ada kekurangan jika anda ingin melihat halaman saya lagi bisa masuk ke halaman utama saya, atau anda juga bisa melihat halaman Html, Css, ataupun halaman Javascript. Oke sekian dari saya wassalamualaikum wr.wb.
