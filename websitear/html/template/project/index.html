Tentang Saya

Halo! Saya adalah pembuat website ini, dan website ini adalah tempat saya bereksperimen — menciptakan project-project kecil, mencoba HTML, CSS, dan JavaScript, serta berbagi hasil kerja saya.


Website ini dibuat sebagai wadah belajar dan bereksplorasi. Anda bisa menemukan berbagai hal: mulai dari pembelajaran HTML, CSS, JavaScript, sampai mini-project seperti kalkulator, kompresor foto, dan fitur-fitur sederhana lainnya — sesuai dengan semangat saya untuk ngulik dan bereksperimen.


Terima kasih sudah berkunjung. Semoga Anda menikmati hasil karya saya. Jika Anda tertarik, jangan ragu untuk hubungi saya atau lihat project saya di bagian Proyek. Selamat berselancar!

Proyek
