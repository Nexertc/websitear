Tentang CSS

Assalamualaikum wr.wb. Pada halaman ini saya ingin memberikan penjelasan tentang bahasa stylesheet yang bernama CSS. Terimakasih telah mengunjungi situs saya ini dan semoga penjelasan ini dapat membantu memahami dasar-dasar pembuatan tampilan website.

CSS adalah singkatan dari Cascading Style Sheets. Jika HTML adalah kerangka atau struktur bangunan, maka CSS adalah desain, warna, dekorasi, dan gaya yang membuat bangunan tersebut terlihat indah dan nyaman dilihat. CSS bertugas mengatur tampilan elemen-elemen HTML seperti warna teks, ukuran font, jarak antar elemen, posisi, animasi, dan tata letak.

Dengan menggunakan CSS, sebuah website yang awalnya hanya terlihat polos dapat berubah menjadi halaman yang memiliki gaya profesional, rapi, dan menarik. CSS juga mempermudah pengembang web untuk mengatur tampilan seluruh halaman hanya dari satu file stylesheet.

CSS memiliki banyak property yang masing-masing berfungsi memberikan gaya tertentu pada elemen HTML. Berikut beberapa property dasar yang sering digunakan dalam pembuatan tampilan website:

  • color : mengatur warna teks.
  • background / background-color : memberi warna atau gambar pada latar belakang.
  • font-size : mengatur ukuran teks agar mudah dibaca.
  • font-family : memilih gaya huruf seperti Arial, Poppins, atau Roboto.
  • margin : mengatur jarak luar elemen.
  • padding : mengatur jarak dalam elemen.
  • border : memberi garis pada elemen, lengkap dengan ketebalan dan warnanya.
  • width dan height : mengatur ukuran lebar dan tinggi elemen.
  • display : mengatur cara elemen ditampilkan, seperti block, inline, flex, dan grid.
  • position : mengatur posisi elemen, seperti absolute, relative, fixed, atau sticky.
  • flexbox : memudahkan mengatur tampilan agar rapi dan responsif.
  • grid : membuat layout kolom dan baris secara modern.
  • transition : membuat efek perubahan halus ketika elemen berubah.
  • animation : membuat animasi seperti fading, sliding, bouncing, dan lainnya.

Selain property, CSS juga memiliki beberapa cara penulisan, yaitu:

  • Inline CSS — ditulis langsung pada tag HTML menggunakan atribut style="".
  • Internal CSS — ditulis di dalam tag <style> pada bagian <head>.
  • External CSS — ditulis pada file terpisah seperti style.css dan dihubungkan menggunakan <link>. Cara ini adalah yang paling rapi dan direkomendasikan.

Kesimpulannya, CSS adalah komponen penting dalam pembuatan tampilan website. CSS berfungsi untuk mengatur warna, ukuran, layout, animasi, jarak, dan seluruh elemen visual pada halaman web. Dengan CSS, sebuah website yang awalnya sederhana dapat terlihat jauh lebih menarik, modern, dan nyaman untuk digunakan. Kombinasi antara HTML sebagai struktur, CSS sebagai tampilan, dan JavaScript sebagai interaksi akan menghasilkan website yang lengkap dan profesional.
