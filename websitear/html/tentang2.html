Sekolah di jurusan TKJ, tapi memiliki skil RPL?

Assalammualaikum wr.wb. Oke untuk di halaman website saya ini saya ingin menjelaskan tentang pengalaman saya kenapa saya memiliki skil RPL di smk ini padahal saya sekolah di jurusan TKJ. Oke sebelumnya saya ingin mengucapkan terima kasih kepada anda karena sudah mengunjungi website saya ini, semoga anda dan keluarga selalu sehat walafiat.

Oke, jadi kenapa saya bisa memiliki skil coding padahal saya sekolah nya itu di TKJ Teknik Komputer dan Jaringan, jadi dulu pada hari Jum'at saya saya pernah kepikiran saya pernah melihat suatu konten di media sosial yang menjelaskan tentang bagaimana membuat game menggunakan AI dan aplikasi Acode Text Editor berbasis Android.

Jadi di konten vidio itu dia menjelaskan tentang membuat game, mulai dari menuliskan promt di AI dan setelah itu di copy paste di aplikasi Acode nya. jadi setelahnya pada hari Jum'at tepatnya pada semester2 kelas XI saya mempraktekannya dengan menulis promt dan mengcopy pastenya ke aplikasi Acode. Setelah itu saya juga mencoba game-game yang lain lainnya juga walau semua itu copy paste dari AI.

Dari situ saya mulai suka dengan coding dan mulai coding dengan yang dasar seperti HTML CSS seperti itu, saya juga mempelajari yang namanya GitHub agar hasil coding website saya itu bisa dilihat oleh audiens.

Selain itu juga saya mempelajari dan mendalami yang lainnya juga seperti mendalami HTML, mendalami CSS, dan mulai dengan belajar JavaScript dasar.
